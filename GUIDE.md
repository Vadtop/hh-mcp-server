# Личная инструкция: как пользоваться HH MCP

---

## Первый запуск (один раз)

1. Открой терминал в папке проекта
2. Запусти авторизацию:
   ```
   python auth_once.py
   ```
3. Введи телефон и SMS-код в открывшемся браузере
4. Готово — сессия сохранена в `.browser_session/`, живёт ~30 дней

**Если MCP вдруг перестал авторизовываться — просто повтори этот шаг.**

---

## Как запустить (каждый раз)

Ничего не надо запускать вручную.

Cline сам стартует MCP сервер при открытии VS Code. Просто пиши в чат Cline.

---

## Каждодневная работа по поиску работы

### Шаг 1 — Найти вакансии

Пиши в Cline:
```
Найди удалённые вакансии по запросам "AI разработчик", "AI интегратор", "внедрение AI", "LLM engineer", "автоматизация n8n". По каждому запросу hh_search с remote=true. Выведи все найденные вакансии списком с ID и ссылками.
```

### Шаг 2 — Оценить и отобрать

```
Для каждой вакансии из списка вызови hh_score_vacancy. Выведи топ-10 по оценке, только те где score >= 65. Покажи: название, компания, зарплата, оценка, ссылка.
```

### Шаг 3 — Откликнуться

На конкретную вакансию (по ID из ссылки — цифры в конце):
```
Откликнись на вакансию [ID] через hh_apply_vacancy
```

Письмо сгенерируется автоматически через AI. Если вакансия требует письмо — оно подставится само.

Если отклик не сработал автоматически — Cline даст текст письма, переходишь по ссылке и вставляешь вручную.

### Шаг 4 — Проверить статусы откликов

```
Покажи мои активные отклики через hh_get_applications
```

## Полезные команды

**Аналитика рынка** (зарплаты, топ навыки, топ компании):
```
Вызови hh_market_analytics с text="AI интегратор"
```

**Что изучить чтобы зарабатывать больше:**
```
Вызови hh_skills_gap с resume_id=[твой ID резюме] и vacancy_text="AI разработчик"
```

**Прогноз зарплаты:**
```
Вызови hh_salary_forecast: current_salary=100000, skills="kubernetes,system design,mlops", timeline_months=6
```

**Сгенерировать письмо без отклика** (посмотреть что получится):
```
Сгенерируй письмо для вакансии [ID] через hh_generate_letter
```

**Узнать ID своих резюме:**
```
Покажи мои резюме через hh_get_my_resumes
```

## Мониторинг ответов (автоматически)

Чтобы получать уведомления в Telegram когда работодатель ответил:
```
Запусти мониторинг через hh_start_monitor с interval=300
```

Проверить прямо сейчас:
```
Вызови hh_check_monitor
```

Остановить:
```
Вызови hh_stop_monitor
```

## Частые вопросы

**Нужно ли запускать что-то каждый день?**
Нет. Просто открываешь VS Code с Cline — MCP стартует сам. Пишешь запрос — получаешь результат.

**Сессия протухла — что делать?**
Запусти `python auth_once.py` в терминале проекта. Займёт 2 минуты.

**Отклик не сработал автоматически?**
Cline даст готовый текст письма. Переходишь по ссылке на вакансию, нажимаешь "Откликнуться", вставляешь текст.

**Как найти ID вакансии?**
Это цифры в конце ссылки: `hh.ru/vacancy/131782229` → ID = `131782229`

**Как найти ID резюме?**
Спроси Cline: `Покажи мои резюме через hh_get_my_resumes` — там будут ID и ссылки.

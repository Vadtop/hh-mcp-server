# 🏗️ HH.ru MCP Server v2 — Архитектура и Roadmap

## 📋 Описание

**AI-powered карьерный ассистент** для hh.ru через **Playwright** (автоматизация браузера).

> **Почему Playwright?** С 15 декабря 2025 API для соискателей закрыт.
> Решение — эмуляция действий реального пользователя через браузер.

### Возможности:
- 🔍 Поиск вакансий через парсинг сайта hh.ru
- 📬 Автоотклики с AI-генерацией сопроводительных писем
- 📄 Просмотр и управление резюме
- 🤖 AI-скоринг релевантности вакансий (0-100)
- 📊 Аналитика рынка труда
- 🎯 AI Career Advisor — анализ навыков, рекомендации
- 📈 Мониторинг ответов на отклики
- 🔔 Уведомления (Console/Telegram)

## 🌐 Playwright Browser Engine

### Авторизация
```
1. Открыть hh.ru/login
2. Ввести номер телефона (через input())
3. Получить SMS-код
4. Ввести код → сессия сохраняется
5. Cookies сохраняются между запусками
```

### Парсинг вакансий
```python
# data-qa селекторы (стабильные)
search_results = page.locator('[data-qa="vacancy-search-results"]')
vacancy_cards = page.locator('[data-qa="search-result-item"]')
apply_button = page.locator('[data-qa="vacancy-response"]')
```

### Анти-детект меры
- Рандомные паузы (2-5 сек между действиями)
- Плавный скролл страницы
- Рандомизация User-Agent
- Обработка капчи (ручная через input())
- Лимит: ~200 откликов/день

### Обработка ошибок
- Модальные окна с вопросами → пропуск
- Обязательное письмо → используем AI-генерацию
- Блокировка → остановка + уведомление

## 📊 Мониторинг ответов на отклики

### Статусы откликов:
- `awaiting_response` — ожидает ответа от работодателя
- `response_received` — получен ответ
- `invited` — приглашение на собеседование 🎉
- `refused` — отказ ❌
- `offer` — предложение работы 🎊

### Логика мониторинга:
```
1. Периодическая проверка (каждые N минут)
2. Парсинг страницы "Мои отклики"
3. Сравнение с предыдущим состоянием
4. Если изменения → уведомление
5. Генерация отчёта (статистика откликов)
6. Сохранение истории в JSON/SQLite
```

### Уведомления:
- Console (по умолчанию)
- Telegram (опционально)
- Email (опционально)
- Webhook (для интеграций)

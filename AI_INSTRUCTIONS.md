# Инструкция для AI ассистента (Qwen Code, Claude, и др.)

## 📖 Что это за проект
HH MCP Server v2 — сервер для работы с hh.ru через AI. 19 MCP инструментов: поиск вакансий, управление резюме, автоотклики, AI-скоринг, карьерный советник.

## 🔑 Как использовать MCP инструменты
1. MCP сервер: `c:\portfolio_2026\hh_mcp_server_v2\src\main.py`
2. Конфигурация: `c:\portfolio_2026\.qwen\settings.json`
3. При старте новой сессии — MCP сервер подключается автоматически
4. Если инструменты не видны — перезапусти сессию (новый чат)

## 📁 Ключевые файлы
- `src/main.py` — MCP сервер (FastMCP)
- `src/web_app.py` — Веб-дашборд (FastAPI, порт 8000)
- `src/config.py` — Конфигурация
- `PROJECT_CONTEXT.md` — Полный контекст проекта
- `logs/` — Директория для логов

## 🚀 Типичные запросы
| Запрос пользователя | MCP инструмент |
|---|---|
| "Найди Python разработчика в Москве 200к" | `hh_search` |
| "Оцени вакансию №1" | `hh_score_vacancy` |
| "Откликнись на вакансию с AI письмом" | `hh_generate_letter` + `hh_apply_vacancy` |
| "Покажи мои резюме" | `hh_get_my_resumes` |
| "Есть ответы на отклики?" | `hh_get_applications` |
| "Проанализируй рынок" | `hh_market_analytics` |
| "Посоветуй карьеру" | `hh_career_advisor` |

## ⚠️ Известные проблемы
- API hh.ru не поддерживает поиск с фильтром по зарплате — только Playwright парсинг
- Авторизация обязательна (телефон + SMS)
- Сессия сохраняется в `.browser_session/`

## 📝 Логи
При каждом запуске пиши в `logs/session.log`:
- Дату и время запуска
- Что делал (поиск, отклик, оценка и т.д.)
- Результат (сколько вакансий найдено, ошибки и т.д.)
